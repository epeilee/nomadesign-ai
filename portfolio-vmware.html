About this Project

虛擬機管理者介面設計 VMware Management Console Interface。客戶為澳洲電信科技商 Ethan Group，主要業務為虛擬機軟體開發維護，最大客戶為 Melbourne Airport。

工作別包含執行 Dashboard、Form Management、Finance Management、Service Request、My Tickets、My Documents等專案頁面規劃，包含客戶需求 User Story、Design System、定義使用者權限、規劃操作流程 User Flow，及視覺延展 Mockup / Prototype 設計。

RWD 規劃：各裝置比重為 Desktop(60%) / Tablet(30%) / Mobile(10%)。
